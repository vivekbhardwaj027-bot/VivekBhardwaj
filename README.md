# Vivek Bhardwaj – Portfolio Website

Premium freelance web developer portfolio focused on healthcare and local business clients.

## 🚀 How to Deploy on GitHub Pages

### Step 1: Upload files
Upload ALL files to your GitHub repository:
- `index.html`
- `style.css`
- `script.js`
- `README.md`

### Step 2: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click **Settings** → **Pages**
3. Under "Source", select **Deploy from a branch**
4. Choose **main** branch → **/ (root)** folder
5. Click **Save**

Your site will be live at:
`https://vivekbhardwaj027-bot.github.io/`

(It may take 1–2 minutes to go live after first deploy.)

## ✅ Customizations To Do

1. **Add Your Photo**
   - Replace the photo placeholder in the Hero section
   - Add `<img>` tag inside `.photo-placeholder` div
   - Recommended size: 280 × 340px, format: WebP or JPG

2. **Update Portfolio Projects**
   - Replace `href="#"` in portfolio cards with your real project URLs
   - Update project titles, descriptions, and technology tags

3. **Add Real Testimonials**
   - Replace placeholder testimonials with real client quotes and names

4. **Google Maps**
   - Replace the map placeholder div with an embedded Google Maps iframe

5. **Form Backend (Optional)**
   - The form currently opens WhatsApp — this works great for direct client contact
   - For email submissions, connect to Formspree (https://formspree.io) — free tier available
